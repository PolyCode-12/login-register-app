# Login & Register App

This is a simple authentication project where users can register and login.

## Technologies
- HTML
- CSS
- JavaScript
- Node.js
- Local JSON database

## Features
- User registration
- User login
- Email and password validation
- Local server support

## How to run

1. Clone the repository
2. Install dependencies

npm install

3. Start the server

node api.js

4. Open index.html in browser
